Usage

Put this url into your browser:

127.0.0.1:PORT/?goto=http://example.com

Where PORT is the first port number you chose on launching the proxy.

Program Limitations

This proxy supports single-document requests with HTTP protocol only. The target url must be specified with

?goto=http://your-url-here

directly after the `127.0.0.1:PORT/`

NOTE: It takes a few seconds to load.

NOTE: It's best to change your port numbers each program execution. It *usually* works without doing this, but not always. The likely reason that it sometimes doesn't work is because the browser has cached the page and it is no longer fetched with a normal HTTP request. Other network and socket errors can sometimes occur as well.

Some urls you can use:

  • http://127.0.0.1:PORT/?goto=http://momogariya.com
  • http://127.0.0.1:PORT/?goto=http://gavinfielder.com
  • http://127.0.0.1:PORT/?goto=http://gavinfielder.com/style.css
  • http://127.0.0.1:PORT/?goto=http://gavinfielder.com/resume.js
